Considere um polinﾃｴmio $P_{n}(x)$ de grau $n$. Defina a seguinte sequﾃｪncia de polinﾃｴmios:
	$g_{0}(x) = P_{n}(x)$
	$g_{1}(x) = P'_{n}(x)$
e em geral, definimos o termo $g_k(x)$ como sendo resto da divisﾃ｣o $-\frac{g_{k-2}(x)}{g_{k-1}(x)}$ 
**I.e**: $g_0/g_1$ = $q_2$, o resto da divisﾃ｣o $r_2$ = $-g_2$, isso de k = 0 atﾃｩ n, sﾃｳ parando quando $g_k$ for um uma constante.

A sequencia $g_{0}(x), g_{1}(x),.., g_{k}(x), ...$ ﾃｩ chamada de sequﾃｪncia de Sturm.

Seja $\alpha\in R$ e $盪ｽ({\alpha})$ o numero de variaﾃｧﾃｵes de sinais da sequﾃｪncia de Sturm obtida **$(g_{k}(\alpha))_{k=1}^{n}$**

# Teorema de Sturm
Sejam $-\infty \leq \alpha \leq \beta \leq +\infty$ . Se $P_{n}(\alpha)\neq 0$ e $P_{n}(\beta) \neq 0$ entﾃ｣o o numero de zeros reais distintos de $P_{n}(x)$ no  intervalo $[\alpha, \beta]$ ﾃｩ exatamente 
> $盪ｽ (\alpha) - 盪ｽ(\beta)$

# Mﾃｩtodo de Newton para polinﾃｴmios
> lembrando que $x_{k+1}= x_{k}- \frac{f(x)}{f'(x)}$

## algoritmo de Briot-Ruffini-Horner
dado 
$P_{n}(x) = a_{n}x^2+.....a_0$ 
$= \sum^{n}_{i=0} a_{i}x^{i}$

e $y\in R$, queremos um algoritmo para calcular $P_{n}(y)$ e $P'_{n}(x)$ 
Sabemos que:
 $P_{n}(x) = (x-y)Q_{n-1}(x) + b_{0}$ 

Seja:
$Q_{n-1}(x) = \sum^{n}_{i=1}b_{i}x^{i-1}$

Entﾃ｣o, 
$\sum^{n}_{i=0}a_{i}x^{i-1} = (x-y). \sum^{n}_{i=1}b_{i}x^{i-1} +b_0$
$= \sum^{n}_{i=1}b_{i}x^{i} - \sum^{n}_{i=1}b_{i}x^{i-1} +b_0$
$\sum^{n}_{i=1}b_{i}x^{i-1} = b_{1}y+b_{2}x +b_{3}yx^{2}+...+b_{n}yx^{n-1}$

$= \sum^{n-1}_{i=0}b_{i+1}yx^{i}$

$\sum^{n}_{i=1} b_{i}x^{i}+b_{0} = b_{n}x^{n}+b_{n-1}x^{n-1} +...+b_{1}x + b_{0}$
$= b_{n}x^{n}+ \sum^{n-1}_{i=0}b_{i}x^{i}$

Para calcular $P'_n(x)$, observe que $= Q_{n-1}(x) + (x-y) Q'_{n-1}(x)$
Logo, $P'_{n(y)}= Q_{n-1}(y)$.

Usando o mesmo artifﾃｭcio para calcular $P_n(y)$
$Q_{n-1}(x)= (x-y)R_{n-2}(x) +c_{1}$, onde $c_{1} = Q_{n-1}(y)=P'_n(y)$

![[briout-houfini-2]]

Com isso, podemos escrever o mﾃｩtodos de Newton ajustado para polinﾃｴmios da seguinte maneira:

Fixada uma aproximaﾃｧﾃ｣o inicial $x_{0}$ para o zero $\xi$ de $P_{n}(x) = ...$
para cada $x_{k}$ obtido na sequencia -> $P_{n}(x_{k})$ e $P'_{n}(x_{k})$
- $y = x_k, b_n = n, c_n = b_n$
# $\implies x_{k+1}= x_{k}- \frac{b_{0}}{c_{0}}$
Se temos uma precisﾃ｣o $\epsilon > 0$, entﾃ｣o, em cada, passo, devemos verificar se  $|x_{k+1} - x_{k}| = |-\frac{b_{0}}{c_{1}}| < \epsilon$ e repetimos o processo para $y = x_k+1$
